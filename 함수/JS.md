## 함수의 기본 형태

### 익명 함수

함수는 코드의 집합을 나타내는 자료형

    function () { }

<br>

    //  변수 생성

    const 함수 = functon () {
        console.log('함수 내부의 코드 ...1')
        console.log('함수 내부의 코드 ...2')
        console.log('함수 내부의 코드 ...3')
        console.log('') }

    // 함수 호출
    함수 ()
    함수 ()

    // 출력
    console.log(typeof 함수)
    console.log(함수)
